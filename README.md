
# Hello D Project: Quick Start

This project lets you build and run a simple D program in Visual Studio Code (VSCode).

---

## 1. Run Without Debugger

1. Open VSCode in this project folder.
2. Select `app.d` in the Explorer so you can see the code in the editor.
3. Build the program:
	- Press `Ctrl+Shift+B` (or select a build task from the Command Palette).
4. In the VSCode Terminal, run:
	```sh
	./hello_d
	```
5. You should see:
	```
	Hello, World!
	```

---

## 2. Run With Debugger

1. Make sure `app.d` is open in the editor.
2. Press `F5` to start debugging.
3. The program will run and output will appear in the Terminal pane.
4. You can set breakpoints in `app.d` if you want to step through the code.
